- 自己写的两数之和：
	- ```xml
	  class Solution {
	      fun twoSum(nums: IntArray, target: Int): IntArray {
	          val hashMap = hashMapOf< Int, Int >();
	          for (i in nums.indices){
	              if( target-nums[i] == hashMap.key){
	                  return i, hashMap.key 
	              }else{
	                  hashMap.key = nums[i];
	                  hashMap.value = i;
	              }
	          } 
	      }
	  }
	  ```
